# Cloud9 RFID SelfCheck — Model Workbench (demo)

Static, self-contained review artifact. No build step, no external calls; state in
`localStorage`. Open `index.html`. Hosted at
`https://chrisjohnsonenvisionware.github.io/c9selfcheck-demo/` and (on VPN)
`https://celsus.envisionware.com/c9selfcheck`.

Surfaces: landing (`index.html`), kiosk (`kiosk/`), admin console (`admin/`),
backend settings console (`backend/`). Design docs under `docs/`.
